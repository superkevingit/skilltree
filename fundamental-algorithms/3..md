# 时间复杂度

<table>
  <tr>
    <td>时间复杂度</td>

    <td>常数级</td>

  <tr>
    <td>theta(lgn)</td>
    <td>对数级</td>
    <td>二分搜索</td>
  </tr>
  <tr>
    <td>theta(n)</td>
    <td>线性级</td>
    <td>列表的遍历</td>
  </tr>
  <tr>
    <td>theta(nlgn)</td>
    <td>线性对数级</td>

  </tr>
  <tr>
    <td>theta(n^3)</td>
    <td>立方级</td>
    <td>Floyd-Warshall算法</td>
  </tr>
  <tr>
    <td>O(n^k)</td>
    <td>多项式级</td>
    <td>

    </td>
  </tr>
